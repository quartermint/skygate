    **Step 1: Create nftables.go** (platform-independent parsing logic).
    - Constants: `nftFamily = "inet"`, `nftTable = "skygate"`, `nftDeviceSet = "device_counters"`, `nftAllowedSet = "allowed_macs"`.
    - JSON parsing types for `nft -j list set` output:
      ```go
      type nftResult struct {
          Nftables []json.RawMessage `json:"nftables"`
      }
      type nftSetElem struct {
          Elem struct {
              Val     string `json:"val"`
              Counter struct {
                  Packets uint64 `json:"packets"`
                  Bytes   uint64 `json:"bytes"`
              } `json:"counter"`
          } `json:"elem"`
      }
      ```
    - `func ParseNftCounters(data []byte) (map[string]uint64, error)` -- parses JSON output from `nft -j list set inet skygate device_counters`. Iterates over Nftables array, attempts to unmarshal each as nftSetElem, collects MAC->bytes.
    - `type DeviceCounters struct { Previous map[string]uint64; Current map[string]uint64 }`
    - `func ComputeDeltas(prev, curr map[string]uint64) map[string]uint64` -- for each MAC in curr: if MAC in prev and curr >= prev, delta = curr - prev; if MAC in prev and curr < prev (counter reset), delta = curr; if MAC not in prev (new device), delta = curr.

    **Step 2: Create nftables_linux.go** with `//go:build linux` tag.
    - `func ReadPerMACCounters() (map[string]uint64, error)` -- executes `nft -j list set inet skygate device_counters`, passes output to ParseNftCounters.
    - `func AddAllowedMAC(mac string) error` -- executes `nft add element inet skygate allowed_macs { MAC_ADDR timeout 24h }`.
    - `func RemoveAllowedMAC(mac string) error` -- executes `nft delete element inet skygate allowed_macs { MAC_ADDR }`.
    - `func IsAllowedMAC(mac string) (bool, error)` -- executes `nft -j list set inet skygate allowed_macs`, parses JSON, checks if mac is in set.
