       - Import: "context", "fmt", "gopkg.in/yaml.v3", "net", "os", "os/signal", "time"
       - Define `Config` struct: `BypassDomains []string \`yaml:"bypass_domains"\``

       - Define `updateNftSetStub(ips []string) error` (non-linux, build tag `//go:build !linux`) — returns nil (no-op for dev)
       - `main()`: loads config from `/data/skygate/bypass-domains.yaml` (overridable via `-config` flag), enters loop: resolveDomains -> updateNftSet, sleep 60s. Handle SIGINT/SIGTERM for clean shutdown via context.
