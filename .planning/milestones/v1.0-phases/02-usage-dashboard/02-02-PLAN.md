<interfaces>
<!-- API contract from RESEARCH.md that the frontend consumes -->

SSE Endpoint: GET /api/events
  - Event "bandwidth": JSON {ts: int64, devices: {mac: bytes_per_sec}, total_bps: uint64}
  - Event "devices": HTML fragment for device table swap
  - Event "cap-status": HTML fragment for plan cap progress bar
  - Event "savings": HTML fragment for dollar savings display
  - Event "chart-data": JSON {labels: string[], values: number[]} for Chart.js update
  - Event "categories": JSON {labels: string[], values: number[]} for pie chart update

REST Endpoints:
  - GET /api/stats/devices -> [{mac, name, bytes_total, top_domain, pct}]
  - GET /api/stats/domains -> [{domain, category, query_count}]
  - GET /api/stats/savings -> {blocked_queries, estimated_bytes_saved, dollar_amount}
  - GET /api/settings -> {plan_cap_gb, billing_cycle_start, overage_rate_per_mb}
  - PUT /api/settings -> accepts JSON {key: value} pairs
  - POST /api/captive/accept -> accepts {mac: string} in body, returns 200

Dashboard layout per D-05 (top to bottom):
  1. Plan cap usage bar with dollar savings (D-19, D-17)
  2. Real-time bandwidth graph (D-03)
  3. Per-device table (D-02)
  4. Category pie chart (D-04)

        // Pre-populate form with saved settings on page load.
        // GET /api/settings returns JSON: {plan_cap_gb, billing_cycle_start, overage_rate_per_mb}
        document.addEventListener('DOMContentLoaded', function() {
            fetch('/api/settings')
                .then(function(r) { return r.json(); })
                .then(function(settings) {
                    var fields = ['plan_cap_gb', 'billing_cycle_start', 'overage_rate_per_mb'];
                    fields.forEach(function(key) {
                        var el = document.getElementById(key);
                        if (el && settings[key] !== undefined) {
                            el.value = settings[key];
                        }
                    });
                })
                .catch(function(err) {
                    console.warn('Could not load settings:', err);
                });
        });
        </script>
    </body>
    </html>
