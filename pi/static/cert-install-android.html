Download Certificate

Download Certificate
2

Install Certificate

When prompted, select CA Certificate and confirm the installation.

You may need to enter your device PIN or pattern to proceed.

Important: Max Savings optimizes browser traffic (Chrome, Firefox, Edge). Native apps use certificate pinning and will continue using DNS-only savings regardless of this setting.
